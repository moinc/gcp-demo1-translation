- Go to the Developer console at: https://console.developers.google.com
- Create your own project (you can use the same project for both demo's)
- Go to APIs & Auth, select API's and turn on the Translate API.
- Go to Credentials and create a new Public Key. Choose Browser Key, 
you can leave the referers box empty.
- Now copy the Project-ID and Public Key to Main.java
- Enjoy.
